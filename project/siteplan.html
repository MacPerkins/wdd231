Website Planning Document

Site Name:

The name of the site will be "Hooked: Utah Fly Fishing with Mac".

Site Purpose:

The purpose of this site is to share information about fly fishing in Utah, including my favotire tips, techniques, and locations.

Scenarios:

1. A user is looking for local fly fishing spots in Utah and finds a list of recommended locations.

2. A user wants to learn about the basics of fly fishing and finds a beginner's guide on the site.

Color Scheme:

The color scheme for the site will be Utah inspired and water colored:

  • Body Background Color: Light Beige (#ece7e0b2)
  • Headings: River Blue (#468fafc9)
  • Sections Background Color: Light Blue (#61c2ec85)

Typography:

The primary font for headings will be "Playfair Display" from Google Fonts. For body text, I will use "Lora" for easy readability.
